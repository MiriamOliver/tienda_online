# tienda_online
## Para Back (Posicionarte en la carpeta back)

Actualizar dependencias:

    npm install
    
Lanzar el servidor:

    nodemon app
    
## Para Front (Posicionarse en la carpeta front)

Actualizar dependencias:

    npm install

Iniciar el cliente:

    ng serve -o

## Base de Datos
Crear una base de datos llamada tienda_online <br>
Importar el archivo tienda_online.sql <br>
